# Claude Agent для Android

ИИ-агент, который управляет твоим Android-телефоном через Accessibility API.
На каждом шаге снимает состояние экрана (UI-дерево + скриншот), отправляет в LLM
через OpenRouter, получает следующее действие (тап / ввод / свайп / назад)
и выполняет его. Цикл повторяется, пока агент не отчитается, что задача
выполнена.

## Что внутри

- **Kotlin + Jetpack Compose** — UI с настройками, полем ввода задачи и логом
- **AccessibilityService** — чтение экрана и выполнение жестов
- **OpenRouter API** (OpenAI-совместимый формат) — выбор любой модели
  (Claude, GPT, Gemini, Llama, DeepSeek)
- **Vision + UI tree** — модель получает и скриншот, и структуру элементов

## Как получить APK без своего компьютера

1. **Создай аккаунт на GitHub** (если ещё нет).
2. **Создай новый пустой репозиторий**, например `claude-agent-android`.
3. **Загрузи туда все файлы этого проекта** — это можно сделать прямо
   в браузере через кнопку "Add file → Upload files" (просто перетащи
   все папки и файлы).
4. После пуша в `main` запустится GitHub Actions (вкладка **Actions** в репо).
   Сборка занимает 3–5 минут.
5. Когда зелёная галочка появится — открой завершённый workflow,
   спустись вниз до раздела **Artifacts** и скачай `claude-agent-debug-apk.zip`.
6. Распакуй — внутри `app-debug.apk`. Перекинь его на телефон и установи
   (нужно разрешить установку из неизвестных источников).

## Настройка на телефоне

1. Получи API-ключ на https://openrouter.ai (вкладка Keys). Положи на счёт
   несколько долларов — этого хватит надолго для экспериментов.
2. Открой приложение Claude Agent → шестерёнка справа сверху →
   введи API-ключ и выбери модель.
3. Включи Accessibility-службу:
   **Настройки Android → Спец. возможности → Установленные службы →
   Claude Agent → Включить**.
   Подтверди предупреждение системы.
4. Вернись в приложение, введи задачу и нажми **Запустить**.

## Примеры задач для пробы

- `Открой Калькулятор и посчитай 137 * 24`
- `Найди в настройках раздел "Звук" и покажи громкость уведомлений`
- `Открой браузер, найди погоду в Москве`
- `Открой Заметки и создай заметку "Купить молоко"`

## Безопасность

- Accessibility-служба видит **весь** UI на экране — включая чужие приложения,
  пароли в полях ввода, личные сообщения. Включай агента осознанно.
- API-ключ хранится локально в DataStore приложения.
- Скриншоты и UI-дерево уходят на серверы OpenRouter и выбранной модели.
  **Не запускай агента в банковских/мессенджерах/при работе с
  чувствительными данными.**
- Для экспериментов лучше создать отдельный гугл-аккаунт на телефоне.

## Известные ограничения MVP

- Простой одношаговый цикл без долгосрочной памяти.
- Скриншоты доступны только на Android 11+.
- Иногда модель выдаёт невалидный JSON — будет видно в логе.
- Нет параллельного выполнения задач, только одна за раз.
- Лимит 20 шагов — после этого агент остановится.
